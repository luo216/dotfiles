## 必备软件

| 软件名称                     | 用处                           |
|------------------------------|--------------------------------|
| openssh                      | ssh服务没有自带                |
| paru                         | aur helper                     |
| dwm                          | 简单的窗口管理器               |
| st                           | 简单的终端                     |
| surf                         | 简单的浏览器                   |
| rofi                         | 软件选择器                     |
| rofi-chinese                 | 带中文补丁的rofi               |
| rofi-polkit-agent-git        | 在需要输入密码提权时候唤出rofi |
| lightdm                      | 显示管理器                     |
| lightdm-gtk-greeter          | 输入框                         |
| lightdm-settings             | 显示管理器设置(autologin)      |
| lightdm-gtk-greeter-settings | 输入框设置(theme)              |
| tmux                         | 终端复用（效率不行）           |
| nvim-0.10                    | 文本编辑器                     |
| lazyvim                      | 预配置框架                     |
